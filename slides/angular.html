## Angular two-way binding Two steps: 1. Find a change 2. Update the DOM
## Step 1: Angular `scope` * 'Glue' between controller and view * 'Watches' for changes in the data and triggers updates
## Step 2: Angular `compile` * Find relevant DOM nodes * Insert `scope` data to produce a view

## Templating Simply `HTML` with 'holes' Can be either inline or in a separate `.html` file ---- `{{variable}}` `<span ng-bind="variable"></span>`

Routing with templates

.when('/home', {
  template: '{{variable}}'
}

.when('/home', {
  templateUrl: 'pages/home.html'
}

Routing with controllers

myModule.controller("MyController",
    function() {...}
);

.when('/home', {
  controller: 'MyController',
  templateUrl: 'pages/home.html'
}
